# 🚀 ATS Resume Analyzer

A web app that analyzes resumes and provides score, skills, and suggestions.

## 🔥 Features
- Upload Resume (PDF)
- Analyze Skills
- Resume Score (ATS)
- Suggestions for improvement

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js + Express
- File Upload: Multer
- PDF Parsing: pdf-parse

## ▶️ How to Run

### Client
cd client
npm install
npm run dev

### Server
cd server
npm install
node index.js
